Casa del lago: En este proyecto se intento crear una relacion entre el interior del espacio y el exterior a traves de ese gran ventanal, combinando y entrelazando materiales de forma tal que al estar dentro te sientas parte del entorno que te rodea.

Habitacion principal: Render final de remodelacion, con el objetivo de modernizar este espacio de la casa, dandole amplitud e iluminacion.

Sala de estar: En este proyecto el cliente hizo enfasis en la utilizacion del color verde como color principal utilizando colores neutros para resaltar el mismo.

Departamento primer piso: En este caso nos vimos frente a un departamento de 45 m2 al cual le faltaba luz natural y espacios abiertos. Por esto mismo se utilizaron colores frios, para dar sensacion y contacto de naturaleza y decidi pintar tanto paredes como suelo de blanco para dar amplitud.

Sala de reuniones: Es un espacio desarrollado para una oficina donde se utilizaron materiales calidos y oscuros para generar intimidad.
